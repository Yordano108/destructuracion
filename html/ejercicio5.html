Ejercicio 05

Objetos Anidados

Acceder y desestructurar claves que se encuentran dentro de sub-objetos internos o ramas estructurales profundas en un único paso.

Datos de Entrada: 
const cliente = {
    id: "CLI-992",
    nombre: "Laura",
    direccion: {
        ciudad: "Barcelona",
        codigoPostal: "08001",
        pais: "España"
    }
};
Destructuración Aplicada: 
const { nombre, direccion: { ciudad, pais } } = cliente;
Resultado esperado: Laura Barcelona España
Abre la consola (F12 o FN + F12) para confirmar el resultado en tiempo real.
