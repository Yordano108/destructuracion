Ejercicio 04

Renombrar Variables

Mapear propiedades con nombres en inglés provenientes de servicios web a alias legibles en español usando la sintaxis de dos puntos (:).

Datos de Entrada: 
const vacante = {
    title: "Frontend Dev",
    salary: 45000,
    remote: true
};
Destructuración Aplicada: 
const { title: titulo, salary: salario } = vacante;
Resultado esperado: "Frontend Dev" 45000
Abre la consola (F12 o FN + F12) para confirmar el resultado en tiempo real.
